Voici le manuel de notre application.

Lors du lancement de l'application vous allez arriver sur un menu, vous avez alors deux options :

1) Vous choisissez un filtre avec les chiffres de 1 à 8.
ou
2) Vous choisissez une autre touche.
   Le programme affiche alors un message d'erreur et vous propose de quitter ou de continuer avec vos touches 0 ou 1.


Si vous entrez une commande valide, le programme vous demande alors de rentrer l'adresse de votre image.
Il faut lui donner une adresse du type : "C:\Users\User\Pictures..."
Cependant, à la fin de cette adresse vous devez également rajouter le nom ainsi que le type de l'image.

Ainsi une adresse valide sur mon PC ressemble à "C:\Users\Quentin\Desktop\TpCpp\lena.jpg"
"lena.jpg" étant le nom et le type de mon image.

Le programme affiche alors deux images :
L'original et la filtrée.

Pour continuer il faut presser une touche.
Le programme vous propose alors de continuer ou de partir avec vos touches 0 ou 1.
Le 0 quitte le programme et le 1 vous renvoie au début.
